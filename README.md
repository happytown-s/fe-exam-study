# FE Exam Study

> 基本情報技術者試験対策 Webアプリ。午前問題のドリル、模試、計算トレーニング、疑似言語トレーニングを網羅した学習ツール。

## Features

- ドリルモード: カテゴリ別に10問/20問/全問をランダム出題
- 模擬試験: 80問・150分の本番シミュレーション（合格ライン60%）
- 間違い復習: 間違えた問題を自動記録し、重点的に復習
- 計算トレーニング: 7トピック・151問の実践的計算問題（基数変換、ビット演算、論理回路など）
- 疑似言語トレーニング: 10トピック・123問のトレース問題（配列、ループ、ソート、再帰など）
- 用語集: 105語のIT関連用語を閲覧
- ダークモード対応
- 進捗のローカルストレージ保存

## Usage
